# 教学实操代码沙盒 (Practice Sandbox)

> **🔒 隐私保护与安全边界说明**  
> 本目录 (`data/materials/Sandbox/`) 是用户在课堂授课和 `/submit` 测验通关中，AI 导师专属布置和修改**编程作业与排错练习题**的实操沙盒。

---

## 核心准则

1. **安全沙盒拦截隔离**：
   AI 导师被工具链底层物理限制，唯一有权使用 `create_file` 和 `replace_file_content` 创建或修改代码的目录，有且仅有本沙盒。

2. **全量隐私屏蔽（禁止上云同步）**：
   隶属于 `data/` 目录，受 `.gitignore` 隐形屏蔽保护，所有练习代码和解题试错痕迹100%仅保存在本地，绝不上云！
