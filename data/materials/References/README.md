# 理论讲义卡片库 (References & Explainers)

> **🔒 隐私保护与数据隔离说明**  
> 本目录 (`data/materials/References/`) 是用户在课堂授课和对话中，AI 导师触发 `<explainer>` 标签时动态生成的**理论知识点备忘录与小结卡片**。

---

## 核心准则

1. **全量隐私屏蔽（禁止上云同步）**：
   本目录隶属于 `data/` 数据空间。在底层的 `.gitignore` 中，`data/` 目录被全量屏蔽隔离。**所有卡片内容仅保存在用户本地计算机中供前端 UI 物料树查阅和本地 RAG 检索，绝不会提交或上传同步至 Cloud 云端工程。**

2. **开发文档与学习数据分离**：
   - **`docs/`**：仅存放系统开发、架构设计、工程规范与测试报告等工程研发文档。
   - **`data/materials/`**：仅存放用户日常学习过程中产生的专属个人数据（包括 `Lessons` 课程、`References` 讲义、`Sandbox` 动手实操代码、`LDRs` 学习记录）。
