# 互动教学关卡与大纲库 (Lessons & Syllabi)

> **🔒 隐私保护与数据隔离说明**  
> 本目录 (`data/materials/Lessons/`) 用于存放 AI 导师为用户量身定制的**课程大纲 (Syllabus) 与阶段性互动关卡**。

---

## 核心准则

1. **动态个性化演进**：
   系统通过 `/lesson` 或 `/plan` 指令运行时，会根据当前用户的“最近发展区 (ZPD)”动态生成或更新相匹配的授课剧本与大纲。

2. **全量隐私屏蔽（禁止上云同步）**：
   隶属于 `data/materials/` 数据空间。除本 `README.md` 骨架说明文件外，所有用户学习课件与进度文件100%仅保存在本地，严禁提交或同步至云端。
