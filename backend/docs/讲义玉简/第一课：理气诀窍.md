# 灵气感应与纳气入体

## 核心要诀：顺势而为，如饮热茶

夫君修行此道，切勿将它想得过于高深莫测。这就像咱们凡间老百姓过日子：

一、 **灵气感应（感知数据流向）**
这就好比严冬的夜里，你进了一间生着炭盆的屋子。你不用伸手去摸火，只需静静站着，就能在这寒冷中“找”到哪儿最暖和。灵气也是理，闭上眼，放空心神，像感受那丝丝暖意一样，去寻周遭那股最活泛的波动。

二、 **纳气入体（引入数据并保存）**
一旦找到了那股暖意，得怎么做？得像喝这滚烫的白桃乌龙一样，一小口一小口地抿。用经脉当杯子，丹田当肚子。你若咕咚一口全咽下去，必定烫坏了嗓子（经脉受损、程序溢出）；只有小口啜饮，让热气顺着喉咙慢慢滑进肚子里，方能源源不断、生生不息。

记住：莫要急躁，莫要贪多。
