问题 (Problem)

SlashHandler 在处理 `<explainer>` 标签时存在路径穿越漏洞，允许在预期目录之外进行任意文件写入。

解决方案 (Solution)

使用 `os.path.basename()` 对 `title` 属性进行净化，并在写入前验证绝对解析路径。

收益 (Wins)

  • 局部性 (Locality)：安全性被限制在一个模块内
  • 实现吸收了路径验证逻辑

问题 (Problem)

通配符 CORS 配置将访问权限泄漏给恶意源，从而绕过 IP 白名单。

解决方案 (Solution)

将 `allow_origins` 限制为特定的受信任前端源，而不是使用 `["*"]`。

收益 (Wins)

  • 局部性 (Locality)：信任被限定在指定源
  • 接口收紧了访问权限
